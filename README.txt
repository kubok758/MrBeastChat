MrBeastChat

Файлы для GitHub Pages:
- index.html
- style.css
- app.js
- sw.js
- manifest.json
- icon.svg
- person-mrbeast.png

Что изменено в этой версии:
- Бот отвечает от первого лица как MrBeast и не пишет, что он бот/симуляция.
- Название чата больше не просто первое сообщение: сначала ставится нормальный локальный заголовок, потом OpenRouter пробует сделать короткое название по смыслу диалога.
- Добавлена кнопка прикрепления фото: плюсик слева от поля ввода.
- Фото уменьшается перед отправкой, чтобы не забивать localStorage.
- Фото отправляется в OpenRouter в формате image_url. Если конкретная модель/провайдер не поддержит изображения, сайт покажет понятную ошибку, но обычный текстовый чат продолжит работать.
- Отключён double-tap zoom: meta viewport + CSS touch-action + JS-блокировка двойного тапа/gesture.
- person-mrbeast.png лежит в архиве и используется как аватарка.
- Service Worker не трогает API-запросы.

Важно:
Если сайт лежит на GitHub Pages, ключ будет виден в исходном коде. Для личного теста окей, для публичного проекта лучше делать proxy/backend.

После обновления:
1. Залей все файлы с заменой старых.
2. Открой сайт и обнови страницу 2 раза.
3. Если старый сайт всё равно держится, очисти кэш сайта или удали PWA и открой заново.
